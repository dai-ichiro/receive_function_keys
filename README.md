# receive_function_keys

F13〜F15などのファンクションキーをフックし、あらかじめ設定されたキー入力やアクションを実行するWindows用バックグラウンドアプリケーションです。

## 概要

一部の多機能マウスやキーボードに割り当てられているF13〜F15などのキー入力を検知し、それらをトリガーとして一連のキーシーケンス（マクロ）を実行します。システムトレイに常駐し、バックグラウンドで動作します。

## 主な機能

- **グローバルキーボードフック**: アプリケーションがフォーカスされていない状態でもキー入力を検知します。
- **カスタマイズ可能なアクション**: `key_config.json` を編集することで、トリガーとなるキーと実行するアクションを自由に変更できます。
- **キーシミュレーション**: 単一のキー入力だけでなく、待機時間を挟んだ連続した入力をシミュレート可能です。
- **軽量動作**: .NET Framework 4.8を使用した軽量なWindows Formsアプリケーションです。

## 使い方

1. `receive_function_keys.exe` を実行します。
2. アプリケーションは起動すると自動的に最小化され、タスクトレイに常駐します。
3. 終了する場合は、タスクトレイのアイコンを右クリックして「終了」を選択してください。

## 設定方法 (`key_config.json`)

実行ファイルと同じディレクトリにある `key_config.json` で動作をカスタマイズできます。

```json
{
  "settings": {
    "logging_enabled": false,
    "key_hold_time": 50
  },
  "actions": {
    "F13": [
      {
        "type": "key",
        "value": "Alt"
      },
      {
        "type": "wait",
        "value": "300"
      },
      {
        "type": "key",
        "value": "H"
      }
    ]
  }
}
```

### 設定項目

- **settings**
    - `logging_enabled`: `true` にすると `debug.log` に実行ログを出力します。
    - `key_hold_time`: キーを押し下げる時間（ミリ秒）を指定します。
- **actions**
    - キー名をキーとし、実行するアクションのリストを値に持ちます。
    - **アクションの種類 (`type`)**
        - `key`: キー入力をシミュレートします。`value` には "A", "Alt", "Ctrl+C" などのキー指定が可能です。
        - `wait`: 指定した時間（ミリ秒）待機します。`value` に数値を指定します。
